# FutManager

App de gestion de equipos de futbol.

## Instalacion

```bash
npm install
```

## Ejecutar

```bash
npm start
```

## Estructura

```
src/
  navigation/   - Navegacion de la app
  screens/      - Pantallas (Login, Home, etc.)
  theme/        - Tema y estilos
assets/         - Iconos e imagenes
```
